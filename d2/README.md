# Git Repo

https://github.com/terrastruct/d2

# Install

```shell
curl -fsSL https://d2lang.com/install.sh | sh -s --
```

# Uninstall

```shell
curl -fsSL https://d2lang.com/install.sh | sh -s -- --uninstall
```

# Command

```shell
d2 --watch --theme 105 --layout "dagre" --host "localhost" --port "22222" test1.d2
```
